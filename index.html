Tools VM Download

Download (Size: ~4.9GiB compressed, ~9.0GiB uncompressed) This machine comes with a variety of tools that we use on a regular basis and come in handy when working on CTFs.

Installation Instructions

This image works best with VirtualBox. After installing, create a new virtual machine for Linux/Ubuntu 64-bit, selecting "Use an existing virtual hard disk file" on the setup screen.

Here's what it should look like. After this, you should assign the VM some more CPU power. In the Settings, under System/Processor, increase the number of processors to at least 2. It should look something like this.

You may also want to install the VirtualBox Guest Additions to be able to expand the screen. You can find the option for it in the Devices menu. Here's an image of what that looks like.
